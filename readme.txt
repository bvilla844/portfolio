# 🧑‍💻 Portafolio Personal - Django

Este es un proyecto web desarrollado con Django que funciona como un portafolio personal. Incluye secciones de presentación, juegos interactivos, gestión de productos, una lista de tareas y un formulario de contacto que guarda información y envía correos.

## 📁 Estructura del Proyecto

### 🔹 1. Portfolio

- Página principal con información personal.
- Sección de proyectos realizados.
- Sección "Sobre mí".
- Formulario de contacto funcional (guarda en base de datos y envía email al propietario del sitio).
- Estilo personalizado con HTML/CSS.


### 🔹 2. Games

- Página con enlaces a juegos:
  - 🎰 Slot Machine
  - 🚢 Battleship
  - (Se pueden abrir en nueva pestaña)

### 🔹 3. Compraventa

- Sección que permite gestionar productos, ordenes, roles de usuario:
  - Electrónicos
  - Tecnologia
  - Alimentos

### 🔹 5. To-Do List

- Lista de tareas con:
  - Agregar tareas
  - Marcar como completadas
  - Eliminar tareas
  - Filtrado por estado

## ⚙️ Tecnologías

- Python 3.x
- Django 4.x
- HTML5 + CSS3 + JS
- Bootstrap (si se usó)
- SQLite3 (modo desarrollo)
- Email (SMTP Gmail)
